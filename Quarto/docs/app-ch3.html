Appendix B — Chapter 3

The connectedness table below is estimated from the Generalized Forecast Error Variance Decomposition (GFEVD) for all 27 reinsurance companies. Each cell \((i,j)\) represents the percentage of firm \(i\)’s forecast error variance attributable to shocks from firm \(j\); diagonal elements indicate own-variance shares; rows sum to “From Others” (total variance received) and columns sum to “To Others” (total variance transmitted). The “Net” row captures net spillover contributions (positive = net transmitter, negative = net receiver).
